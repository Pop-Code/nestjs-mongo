Static Options

Options<K, V>: object
  • Defined in /Users/alexandrehermann/Dev/nestjs/nestjs-mongo/node_modules/dataloader/index.d.ts:79

Type parameters

  • K

  • V

Type declaration

  • Optional batch?: boolean

    Default true. Set to false to disable batching, instead immediately invoking batchLoadFn with a single load key.

  • Optional cache?: boolean

    Default true. Set to false to disable memoization caching, instead creating a new Promise and new key in the batchLoadFn for every load of the same key.

  • Optional cacheKeyFn?: function

    A function to produce a cache key for a given load key. Defaults to key => key. Useful to provide when JavaScript objects are keys and two similarly shaped objects should be considered equivalent.

      • (key: any): any

      • Parameters

        • key: any

        Returns any

  • Optional cacheMap?: CacheMap<K, Promise<V>>

    An instance of Map (or an object with a similar API) to be used as the underlying cache for this loader. Default new Map().

  • Optional maxBatchSize?: number

    Default Infinity. Limits the number of items that get passed in to the batchLoadFn.
